## Overview

Intelligent-fronthaul-network-optimization is a powerful developer tool designed to analyze, visualize, and optimize fronthaul network performance through data-driven insights. It combines topology inference, traffic analysis, and capacity estimation to support network reliability and scalability.

**Why intelligent-fronthaul-network-optimization?**

This project helps network engineers understand complex fronthaul structures and traffic dynamics. The core features include:

- üåø üîç **Topology Inference:** Automatically deduces cell-to-link relationships using correlation and clustering techniques.
- üìä üìà **Traffic Visualization:** Generates detailed plots of link traffic patterns over time, highlighting bottlenecks.
- üîß **Capacity Estimation:** Calculates minimum link capacities considering buffers and peak loads for robust planning.
- üåê **Data Aggregation:** Converts cell-level data into comprehensive link-level insights for performance monitoring.
- üì± **Interactive Dashboard:** Provides an accessible interface for exploring network topology and capacity metrics.

## Getting Started

### Prerequisites

This project requires the following dependencies:

- **Programming Language:** Python
- **Package Manager:** Pip

### Installation

Build Intelligent-fronthaul-network-optimization from the source and install dependencies:

1. Clone the repository:

```bash
git clone https://github.com/Prj-geek/Intelligent-fronthaul-network-optimization
```

2. Navigate to the project directory:

```bash
cd Intelligent-fronthaul-network-optimization
```

3. Install the dependencies:

Using [pip](https://pip.pypa.io/en/stable/):

```bash
pip install -r streamlit_app/requirements.txt
```

### Usage

Run the project with:

Using [pip](https://pip.pypa.io/en/stable/):

```python
python {entrypoint}
```

### Testing
